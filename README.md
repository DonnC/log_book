# LogBook Desktop App

an internship log book desktop app

## Features
- [x] Create quick ToDos
- [x] Delete ToDos
- [x] Add Log Book entry with date and workdone
- [x] View added logbooks in grid / list view
- [x] Generate and print all logbooks
